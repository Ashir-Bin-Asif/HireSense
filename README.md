HireSense: Intelligent Resume Evaluation & Job Hunting Agent

HireSense is an automated AI pipeline built using n8n, Docker, and LLMs (Google Gemini). It ingests applicant resumes from a web form, automatically extracts profile data, executes live job matching queries against real-time job opportunities, and generates a responsive, tailor-made ATS analytics report dashboard.

#Features
* Automated PDF file ingestion and structural text extraction.
* Dynamic live job market discovery via custom HTTP integrations.
* Modular LLM pipeline utilizing 3 Gemini Models for speedy data normalization and deep qualitative job analytics.
* Custom-themed styling implementation for polished frontend report generation.

#Tech Stack
* Workflow Automation: n8n
* AI Models: Google Gemini 3.5 Flash
* API Calling for Job Search: Rapid API
* Frontend Design: HTML, CSS.
